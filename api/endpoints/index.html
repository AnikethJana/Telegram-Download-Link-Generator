    API Endpoints Reference¶

    This page provides detailed documentation for all available StreamBot API endpoints.

    System Information¶

    GET /api/info¶

    Returns comprehensive bot status and configuration information.

    Authentication: None required

    Request: 

    GET /api/info HTTP/1.1
Host: yourdomain.com
Accept: application/json

    "link_expiry_enabled": true,
    "link_expiry_duration_seconds": 86400,
    "link_expiry_duration_human": "24 hours"

    "cors_enabled": true,

    status string API response status (ok or error)
    bot_status string Telegram bot connection status

    features object Enabled features and their configuration

    uptime string Human-readable bot uptime
    server_time_utc string Current server time in UTC ISO format
    totaluser integer Total number of registered users
    github_repo string Repository URL (if configured)

    File Downloads¶

    GET /dl/{encoded_id}¶

    Download files via generated download links. PERFORMANCE OPTIMIZED for speed.

    Authentication: None (uses encoded file IDs for security)

    Security Features: - ✅ Global rate limiting (500 requests/minute) - ✅ Basic input validation (lightweight) - ✅ Essential security headers - ✅ Bandwidth monitoring - ✅ Error handling without information disclosure

    Request: 

    GET /dl/VGhpcyBpcyBhIGZha2UgZW5jb2RlZCBpZA HTTP/1.1
Host: yourdomain.com
Range: bytes=0-1023
User-Agent: Mozilla/5.0 (compatible)

    Response (Success - 200/206): 

    HTTP/1.1 206 Partial Content
Content-Type: application/pdf
Content-Length: 1024
Content-Range: bytes 0-1023/2048576
Content-Disposition: attachment; filename="document.pdf"
Accept-Ranges: bytes
X-Content-Type-Options: nosniff
X-Frame-Options: DENY
Server: StreamBot

[Binary file content]

    Rate Limiting: - Global Limit: 500 downloads/minute across all users - Error Response: HTTP 429 with retry information - No per-client tracking for optimal performance

    Authentication Reference¶

    JWT Token Generation¶

    Use the Telegram bot to generate JWT tokens:

    /token    - Generate 24-hour API access token (admin only)
/revoke   - Revoke all your active tokens (admin only)

    Authentication Methods Summary¶

    Method Security Level Performance Use Case
    JWT Bearer Token High Fast API automation, scripts
    Legacy Token Parameter Medium Fast Simple tools, testing
    Admin IP Access High Fast Server-side access

    Rate Limiting by Endpoint¶

    Endpoint Rate Limit Window Notes
    /dl/* 500/minute Global Performance optimized
    /api/logs 10/minute Per IP Admin endpoint
    /api/security 10/minute Per IP Admin endpoint
    /api/info 30/minute Per endpoint Public endpoint

    Usage Examples¶

    cURL Examples¶

    # Get bot information
curl -X GET "https://yourdomain.com/api/info"

# Get error logs with pagination
curl -X GET "https://yourdomain.com/api/logs" \
  -G \
  -d "token=your_access_token" \
  -d "level=ERROR" \
  -d "limit=50" \
  -d "page=1"

# Download a file
curl -X GET "https://yourdomain.com/dl/encoded_id/filename.pdf" \
  -o "downloaded_file.pdf"

# Download with range request (first 1024 bytes)
curl -X GET "https://yourdomain.com/dl/encoded_id/filename.pdf" \
  -H "Range: bytes=0-1023" \
  -o "partial_file.pdf"

    Python Examples¶

    import requests

# Get bot information
response = requests.get('https://yourdomain.com/api/info')
data = response.json()
print(f"Bot status: {data['bot_status']}")

# Get logs (admin only)
response = requests.get('https://yourdomain.com/api/logs', params={
    'token': 'your_token',
    'level': 'ERROR',
    'limit': 50
})
logs = response.json()

# Download file with progress
def download_file(url, filename):
    response = requests.get(url, stream=True)
    total_size = int(response.headers.get('Content-Length', 0))

    with open(filename, 'wb') as f:
        downloaded = 0
        for chunk in response.iter_content(chunk_size=8192):
            f.write(chunk)
            downloaded += len(chunk)
            progress = (downloaded / total_size) * 100
            print(f"Progress: {progress:.1f}%")

    JavaScript Examples¶

    // Get bot information
async function getBotInfo() {
    try {
        const response = await fetch('https://yourdomain.com/api/info');
        const data = await response.json();
        console.log('Bot info:', data);
    } catch (error) {
        console.error('Error:', error);
    }
}

// Download file with progress tracking
async function downloadFile(url, filename) {
    const response = await fetch(url);
    const contentLength = response.headers.get('Content-Length');
    const total = parseInt(contentLength, 10);

    const reader = response.body.getReader();
    let downloaded = 0;

    while (true) {
        const { done, value } = await reader.read();
        if (done) break;

        downloaded += value.length;
        const progress = (downloaded / total) * 100;
        console.log(`Progress: ${progress.toFixed(1)}%`);
    }
}
